Lab 3: School Districting¶

In this lab, we will investigate school districting.

  • Dataset Overview: Our analysis will make use of 201 census blocks within the jurisdiction of Hope Elementary School District.
  • Assignment Criteria: Each census block will be assigned to a school based on several factors, including the number of elementary students and the fraction of minority race within each block.
  • Optimization Objective: The primary criteria in the assignment process is to minimize travel costs, which can be measured in terms of distance or time, ensuring efficient accessibility for all students within the district.

Through this lab, we aim to gain insights into the complex decision-making involved in the creation and optimization of school districts, considering various socio-demographic factors and logistical considerations!

$C_j$ indicates that it's an array. In that case, different capacity constraint is assigned to each school.

Question 7: Prepare $D_{ij}$ variable for the model. ( 2 pts )¶

$D_{ij}$ represents a two-dimensional array with two subscripts, $i$ and $j$.

Considerations:

  • The first subscript $i$ denotes the row index in the two-dimensional array, corresponding to a block with index $i$.

  • The second subscript $j$ denotes the column index in the two-dimensional array, representing a school with index $j$.

    • For instance, if we have 100 blocks and 10 schools, $D_{ij}$ would be a two-dimensional array (or matrix) of shape (100, 10), consisting of 100 rows and 10 columns.
    • Consequently, $D_{ji}$ would be a matrix of shape (10, 100), containing 10 rows and 100 columns.

  • $D_{ij}$ denotes that given a row index $i$ and column index $j$, a specific value can be retrieved, which represents the distance between the $i$th block and the $j$th school.

    • For example, $D_{12}$ would indicate the distance between the second block and the third school.

Given that, let's create the $D_ij$ variable.

Step 3: Optimization ( Wei et al., 2021 )¶

The model presented in the paper:

\begin{align*} \text{minimize} \quad & \sum_{i=1}^{I} \sum_{j=1}^{J} D_{ij} \cdot S_i \cdot x_{ij} \\ \text{subject to} \quad & \sum_{j=1}^{J} x_{ij} = 1 \quad \text{for } i = 1, \ldots, I \\ & \sum_{i=1}^{I} x_{ij} \cdot S_i \leq C_j \quad \text{for } j = 1, \ldots, J \\ & \sum_{i=1}^{I} (FN_{\text{low}} \cdot S_i - N_i) \cdot x_{ij} \leq 0 \quad \text{for } j = 1, \ldots, J \\ & \sum_{i=1}^{I} (N_i - FN_{\text{high}} \cdot S_i) \cdot x_{ij} \leq 0 \quad \text{for } j = 1, \ldots, J \\ & x_{ij} \in \{0,1\} \quad \text{for } i = 1, \ldots, I \text{ and } j = 1, \ldots, J \end{align*}

Question 9: Construct the School Districting Optimization Model using the provided skeleton code structure. ( 5 pts )¶

In [90]:
model = gp.Model("School Districts")

# Create decision variables
x = model.addVars(I, J, vtype=GRB.BINARY, name="x")

# Set objective
model.setObjective(gp.quicksum(gp.quicksum(x[i,j] * S_i[i] * D_ij[i][j] for i in range (I)) for j in range (J)), GRB.MINIMIZE)

# Add constraints
# Constraint 1
model.addConstrs(gp.quicksum(x[i,j] for j in range(J)) == 1 for i in range(I))
# Constraint 2
model.addConstrs(gp.quicksum(x[i,j] * S_i[i] for i in range(I)) <= C for j in range(J))
# Constraint 3
model.addConstrs(gp.quicksum(x[i,j] * ((FN_low * S_i[i]) - N_i[i]) for i in range (I)) <= 0 for j in range (J))
# Constraint 4
model.addConstrs(gp.quicksum(x[i,j] * (N_i[i] - (FN_high * S_i[i])) for i in range (I)) <= 0 for j in range (J))
# Optimize model
model.optimize()

def evaluate_assignment(col = "curr_assigned"):
    popsums = []
    FNs = []

        FNs.append((1 - wsum/popsum)*100)

# Read the driving origin-destination (OD) matrix from a CSV file
od = pd.read_csv("Data/driving_od_matrix.csv")

# Initialize a matrix D_ij to store driving distances between each block and each school
D_ij = np.ones((len(blocks), len(schools))) * -1

# Populate the D_ij matrix with driving distances from the OD matrix
for o in range(len(od)):
    # Extract the origin and destination IDs from the OD matrix
    i, j = od.loc[o, ["OriginID", "DestinationID"]]
    # Store the driving distance in meters in the corresponding cell of D_ij matrix
    D_ij[i - 1][j - 1] = od.loc[o, "Total_Kilometers"] * 1000

# Check if all elements of the D_ij matrix are non-negative, indicating valid driving distances
(D_ij >= 0).sum() == len(blocks) * len(schools)

# Copy and Paste your optimization model above!
model = gp.Model("School Districts")

# Create decision variables
x = model.addVars(I, J, vtype=GRB.BINARY, name="x")

# Set objective
model.setObjective(gp.quicksum(gp.quicksum(x[i,j] * S_i[i] * D_ij[i][j] for i in range (I)) for j in range (J)), GRB.MINIMIZE)

# Add constraints
# Constraint 1
model.addConstrs(gp.quicksum(x[i,j] for j in range(J)) == 1 for i in range(I))
# Constraint 2
model.addConstrs(gp.quicksum(x[i,j] * S_i[i] for i in range(I)) <= C for j in range(J))
# Constraint 3
model.addConstrs(gp.quicksum(x[i,j] * ((FN_low * S_i[i]) - N_i[i]) for i in range (I)) <= 0 for j in range (J))
# Constraint 4
model.addConstrs(gp.quicksum(x[i,j] * (N_i[i] - (FN_high * S_i[i])) for i in range (I)) <= 0 for j in range (J))
# Optimize model
model.optimize()
